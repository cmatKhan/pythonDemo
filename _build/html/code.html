capital_mod.capitalize(infilename, outfilename)¶

reads the contents of infilename, and writes it to outfilename, but with every word capitalized

note: very primitive – it will mess some files up!

this is called by the capitalize script

Parameters

  • infilename (string) – The file name you want to process

  • outfilename (string) – the name of the new file that will be created

Returns

None

Raises

IOError if infilename doesn’t exist.
